# fence-proof-of-work

这是一个可直接参考的 VisitAPI + Fence 最小闭环样例。

包含：

- 原生商人 `Fence` 的 `.dlg`
- 战局内 `trigger: raid`
- raid 内直接 `accept` / `handover` / `complete`
- 一条原生 `Quest` 结构的上交任务
- 中英 locale
- 背景图
- 任务图标占位文件

## 用途

- 作为“战局内遇到商人并直接接交任务”的最小范式
- 作为 VisitAPI 对话、任务、locale 联动的参考样例

- `assets/backgrounds/FenceBackGround.png` 为对话背景
- `assets/quest-icons/fence-proof-of-work.png` 当前是占位图标，可后续替换成正式 icon
