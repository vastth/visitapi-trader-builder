# 05 常见错误清单

## Quest JSON

- 把 `HandoverItem` 写成 `items`
- 外层 quest id 和内部 `_id` 不一致
- `templateId` 与 `_id` 不一致
- 奖励里把 `_id` 和 `_tpl` 混用
- 自己发明 `Quest.cs` 没有的字段

## DLG

- `.dlg` 文件名不是 trader id
- `trader:` id 与文件名不一致
- 选项跳转到不存在节点
- raid 内主流程仍然依赖 `@tasks`

## TPL

- 用了 Git LFS 指针版 `items.json`
- 用中文物品名直接写进 `target`
- 错把奖励实例 id 当模板 id

## locale

- 少了 `name` / `description` 等必填 key
- 少了 `HandoverItem` 条件 id 对应的 locale key
- 只写中文不写英文，却又要求双语 quest locale

## VisitAPI 运行边界

- 把 `once` 当作 raid 内可靠的一次性开关
- 把 `door` 当作精确碰撞盒
- 假设 raid 内打开对话后战局暂停

## assort

- 没先确认 trader id
- 没先确认 quest 是否已经注册
- 复杂商品树没做 parent/slot/barter 连带检查

## 建议

每次生成新任务前，至少运行：

- `validate_quest_json.py`
- `validate_tpl_refs.py`
- `validate_locale_keys.py`
- `validate_dlg.py`
